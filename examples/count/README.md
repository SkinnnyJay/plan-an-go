# Count example

Minimal plan-an-go example: write the numbers 1–10 to `./test.txt` (one per line) in this folder.

- **Plan:** [PLAN.md](./PLAN.md)
- **Run from repo root:** `npm run example:count` or `./examples/count/run.sh`
- **Behavior:** Log file path is printed first, then pipeline output streams and is appended to that file.

Requires a configured CLI (`claude`, `codex`, or `cursor-agent`). See repo root [README](../../README.md) for setup.
